# 新标的发掘 + 低价替代扫描

## Contract

- **Triggers**: daily-briefing P3.5 / "找便宜替代" / "发掘新机会" / "扫一下板块" / 信号股价 >¥40 或 52w >200%
- **Inputs**: 板块名 + 已知龙头(参考)
- **Outputs**: 3-5 个新标的(代码/价/52w/PE/产业链位置/入选理由),并入 `knowledge/watchlist-sectors.md`
- **Calls**: `bottleneck-analysis`(三维度初筛)、`tool-fallback`(数据失败)
- **Called by**: `daily-briefing`(P3.5)、用户直接调用

**核心原则**:不能只分析 watchlist 已有公司。每次研究 session 必须拓展新标的。

## 触发条件

### A. 自动触发（每次 daily workflow）
- 板块扫描时必发掘 3-5 个未跟踪标的
- 任何信号股价 >¥40 → 强制找便宜替代
- 任何信号 52w 涨幅 >200% → 标"已充分定价"+ 找滞涨替代

### B. 用户主动触发
- "找便宜的"、"低估值"、"被忽视"、"还有什么没涨的"

## 发掘方法

### 1. 同链条替代（首选）
同一供应链环节 → 找市值更小、股价更低（**¥30 以下优先**）、涨幅更小的标的

### 2. 上游/材料端
龙头充分定价时 → 检查上游原材料/零部件供应商

### 3. 港股折价
检查 A 股 / 港股对应标的的估值差（PE 差 5-10 倍）

### 4. 估值过热筛查
52w 涨幅 >200% 自动标"已充分定价风险"，优先 <100% 滞后标的

## 入选条件（加入 watchlist-sectors.md）

- ¥30 以下**优先**（用户偏好，详见 user_risk_preference 记忆）
- 52w 涨幅 <100%（避免追高）
- 逻辑清晰（在产业链关键环节）
- 通过 `bottleneck-analysis` skill 三维度初筛
- **🔴 4 维位置必报**(覆盖 CLAUDE.md "📊 滞涨/认知差 标签强制 4 维校验"):
  - 52w 涨幅 + 区间分位(距高 -X%、距低 +Y%) / 近 20 日 / 近 5 日 / 是否放量跳涨
  - 任一阈值命中(近5日>10% / 近20日>15% / 距52w高<15% / 放量) → **不能贴"滞涨/认知差"标签**,只能标"业绩硬+已启动"

## 验证流程

发现新公司 → Yahoo Finance `get_stock_info` 验证：
- 当前价格
- PE / PB / 市值
- 52w 高低
- marketState（确认 ticker 真实存在 + 在交易）
