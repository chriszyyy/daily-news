# Signal Generation Pipeline

**唯一信号入库通道**。任何即将写入决策日志/positions.md 的信号必须走完此 4 步,一步不过 = 信号丢弃或降级。

## Contract

- **Triggers**: 生成新信号 / 加减仓决策 / 卖出决策 / daily-briefing P3.x
- **Inputs**: 标的代码、当前价、初步逻辑(为什么看好)
- **Outputs**: 完整信号行(技术面+目标-止损+赔率+Kelly+反方+信心+最终仓位)
- **Calls**: `bottleneck-analysis`(验证瓶颈定价潜力)、`tool-fallback`(数据失败时)
- **Called by**: `daily-briefing`、用户直接生成信号时

## 4 道闸(顺序执行,任一失败 → STOP)

### Gate 1: 技术面 — 入场时点

**数据**:Yahoo Finance `get_historical_stock_prices` period=6mo;A股用 `python tools/technical.py`;美股可用 Alpha Vantage 直接拉。

**必算指标**:

| 指标 | 解读 |
|------|------|
| MA20 / MA60 | 趋势 + 金叉/死叉 |
| RSI14 | >70 超买,<30 超卖 |
| MACD (DIF/DEA/柱) | 动能方向 |
| 布林 %B | 波动位置 |

**判定**:

- 🟢 **可入场**:RSI 30-60 + 价 > MA20 (或金叉) + MACD 金叉/柱放大 + %B 20-50
- 🔴 **不要追高**:RSI > 70 / 价远离双线 / MACD 顶背离 / %B > 80
- 🟡 **等右侧**:价 < MA20+MA60 / MACD 死叉柱放大 / %B < 20

**输出**:
```
- 价 ¥X / MA20 ¥Y / MA60 ¥Z → [多/空/震荡]
- RSI14: N / MACD: [金叉/死叉] / %B: M%
- 入场建议: [立即/等回调¥X/暂不]
- 关键支撑: ¥X (距现价 N%)
```

### Gate 2: Kelly 仓位计算

**公式**:
```
赔率 b = (目标价 - 当前价) / (当前价 - 止损价)
Kelly f* = (p×b - q) / b   ; q=1-p
实际仓位 = 半 Kelly = f*/2
```

**工具**:`python tools/kelly.py <price> <target> <stop> <winrate>`

**赔率分级**:

| b | 评价 | 操作 |
|---|------|------|
| > 2 | 优秀 | 半 Kelly 可上 |
| 1-2 | 一般 | 需 >55% 胜率 |
| < 1 / Kelly 负 | 差 | 不下注 |

**胜率参考**:逻辑直接+估值合理 55% / 不确定大 50% / 催化远 45%

**仓位下限**:**最小 ¥3,000**;Kelly 算出 < ¥3,000 → 不买。

### Gate 3: 瓶颈硬约束(覆盖 Kelly)

**必调** `bottleneck-analysis` skill 完成 6 问评分 + 三维度扫描 + 瓶颈定价潜力评级(低/中/高)。

**仓位与确定性匹配**:

| 确定性 | 上限 |
|--------|------|
| 极高(已验证瓶颈+订单+涨价) | 10% |
| 高(瓶颈+部分验证) | 7-10% |
| 中(瓶颈未完全验证) | 5-7% |
| 中低(有逻辑+大弹性) | 3-5% |
| 低(题材) | 0-3% |

**集中度硬约束**:

- 单票 **3%-10%**;>15% **绝对禁止**(实物黄金除外)
- 同主线必须**平铺 3-5 个核心环节**
- **任一标的 >10% → 下次必须减仓**
- **同主线集中度 >40% → 暂停加仓**

**最终仓位 = min(半 Kelly, 硬约束上限)**

### Gate 4: Devil's Advocate(必写)

每条信号必须写**反方观点**,4 类至少覆盖 2 类:

1. **反面论证**:本信号最强的反对论据是什么?
2. **数据矛盾**:有没有数据指向相反方向?
3. **时机风险**:为什么现在不是入场时点?
4. **历史类比**:过去类似情况下结果如何?

**判定**:**反方比正方强 → 信号降级或丢弃**。

## 卖出标准(基于"逻辑验证")

✅ **应卖**:瓶颈消失/需求证伪/竞争变差/估值透支(PE>历史90分位+增速放缓)/主线推翻

❌ **禁卖**:短期波动<10%/情绪恐慌/眼红别人涨/单次业绩低于预期但行业未变/月线 MA20 跌破但季线趋势完整

## 输出格式(写入决策日志/positions.md)

```markdown
### [代码 公司名] 信号

#### Gate 1 技术面
- 价 / MA20 / MA60 / RSI / MACD / %B
- 入场建议: ...

#### Gate 2 Kelly
- 当前 / 目标 / 止损
- 赔率 b / 胜率 p / Kelly f* / 半Kelly K%

#### Gate 3 瓶颈硬约束
- 瓶颈定价潜力: [低/中/高]
- 单票 ≤10% ✅ / 主线 ≤40% ✅ / 非 >10% 状态 ✅
- 上限: X%

#### Gate 4 Devil's Advocate
- 反面论证: ...
- 时机风险: ...
- 结论: 反方<正方 → 入库 / 反方≥正方 → 降级

#### 最终信号行
| 操作 | 标的 | 入场价 | 逻辑 | 反方观点 | 目标/止损 | 赔率 | Kelly | 信心 |
| ... | ... | ¥X | ... | ... | ¥Y/¥Z | b | K% | ⭐⭐⭐⭐ |
```

## 工具失败处理

任意数据获取失败 → 调用 `tool-fallback` skill。绝不跳过 Gate。
