# 基本面深度研究

## Contract

- **Triggers**: "深度研究 XX 公司" / "看一下 XX 基本面" / daily-briefing P3.3 / Kelly 为正进入候选时
- **Inputs**: 标的代码 + 板块归属
- **Outputs**: 6 维度 + G 框架摘要,写入决策日志的"基本面摘要"
- **Calls**: `bottleneck-analysis`(G 框架强制)、`tool-fallback`(数据失败)
- **Called by**: `daily-briefing`(P3.3)、用户直接调用

对每个候选标的(Kelly 为正或进入关注清单),系统性分析 6 维度 + G 框架。

## A. 财报分析

- Yahoo Finance `get_financial_statement`：最近 2-3 季度的 income_stmt + balance_sheet + cashflow
- 关注：营收增速、毛利率变化、经营现金流、资产负债率
- 🚩 红旗：营收增长但现金流恶化、应收账款暴增、有息负债快速攀升

## B. 估值与市场预期

- Yahoo `get_stock_info`：PE/PB/PS、市值、52w 高低、分析师目标价
- Yahoo `get_recommendations`：券商评级近 6 月变化
- 同板块标的横向对比 PE/PB → 识别溢价/折价
- **🔴 4 维位置必报(覆盖 CLAUDE.md "📊 滞涨/认知差 标签强制 4 维校验")**:
  1. 52w 涨幅 + 区间分位(距高 -X%、距低 +Y%)
  2. 近 20 日涨幅%
  3. 近 5 日涨幅%
  4. 是否近期放量跳涨(单日 +5% 且量能 ≥2x)
  - 任一命中阈值(近5日>10% / 近20日>15% / 距52w高<15% / 已放量) → **取消"滞涨认知差"叙事**,赔率 b 上限压到 ≤2x

## C. 股东与资金动向

- Yahoo `get_holder_info`：机构持仓变化、前十大股东、内部交易
- 关注：机构增/减持趋势、大股东质押比例、高管减持

## D. 合作方与供应链定位

- Exa：`"[公司名] 客户 供应商 合作"`、`"[公司名] 供应链"`
- 绘制关键客户/供应商关系
- ⚠️ 客户集中度 >30% 标注警告

## E. 竞争格局与护城河

- Exa：`"[公司名] 竞争对手 市占率"`、`"[板块] 竞争格局"`
- 护城河类型：技术壁垒/客户粘性/规模效应/牌照/专利
- 替代风险：技术路线变化对该公司的威胁

## F. 政策催化与风险

- Exa：`"[板块] 政策 补贴 规划"`
- 即将到来的催化（财报日、行业展会、政策发布窗口）
- 政策风险（监管收紧、出口管制、行业整治）

## 🎯 G. 瓶颈资产 6 问框架（强制）

调用 `bottleneck-analysis` skill 完成：
- 6 问评分（4 题胜率 + 2 题盈亏比）
- 三维度扫描（需求/供给/认知）
- 海外对标估值差对比
- "瓶颈定价潜力"评分：低/中/高

## 输出格式

写入决策日志的"基本面摘要"：

```markdown
### [公司名 代码] 基本面摘要
- A. 财报：[关键数据 + 红旗]
- B. 估值：PE X / PB Y / 较同业 [溢价/折价 Z%]
- C. 股东：[机构/北向变化]
- D. 供应链：[大客户 + 集中度]
- E. 竞争：[护城河 + 威胁]
- F. 政策：[催化 + 风险]
- G. 瓶颈框架：6 问评分 [...] / 瓶颈定价潜力 [低/中/高]
- **一句话结论**：利好 / 中性 / 利空
```

## 工具失败处理

- 美股财报 fallback: `stockanalysis.com/stocks/{ticker}/financials/`
