# 瓶颈资产投资框架

**本 skill 是瓶颈框架的唯一流程来源**。`knowledge/bottleneck-framework.md` 仅保留方法论数据(AI 主线摊开图、海外对标、当前持仓诊断)。

## Contract

- **Triggers**: 单标的研究 / 信号生成 / 卖出决策 / 产业链下钻 / daily-briefing P3.1 / signal-generation Gate 3
- **Inputs**: 标的代码 + 当前价 + 板块/产业链位置
- **Outputs**: 6 问评分表 + 三维度扫描 + 认知差结论 + **瓶颈定价潜力评级(低/中/高)**
- **Calls**: 无
- **Called by**: `signal-generation`(Gate 3)、`fundamental-deepdive`(G 框架)、`discover-new-stocks`(初筛)、`daily-briefing`(P3.1)

## 核心理念

**真正的 10X 机会 = 市场重新理解一家公司在产业链里的位置**

- 原:"普通供应商"/"周期股"(PE 10-20x)
- 后:"AI 基建瓶颈"/"关键节点"(PE 50-100x)
- 估值跳变 + EPS 增长 = **戴维斯双击 → 5x/10x 股**

## 三维度扫描(每个候选必做)

### 维度1:需求
- [ ] 下游订单加速?(YoY 增速、订单可见性)
- [ ] 大客户锁单/付预付款?
- [ ] 行业增速 vs GDP 差距?
- [ ] 真实业务 vs 题材炒作?

### 维度2:供给
- [ ] 交付周期延长?(>12月=严重瓶颈,>24月=极度瓶颈)
- [ ] 行业涨价?(季度>10%=明显瓶颈)
- [ ] 产能扩张周期?(新建厂 2-3 年=供给约束强)
- [ ] 技术/认证壁垒?
- [ ] 上游卡脖子?

### 维度3:认知
- [ ] 当前 PE/PB 估值框架?
- [ ] 海外对标估值?
- [ ] 卖方覆盖度?
- [ ] 新闻/搜索热度?
- [ ] 资金属性(散户/机构/北向)?

## 6 问框架(评分入决策日志)

| # | 问题 | 维度 | 评分 |
|---|------|------|------|
| 1 | 所在产业真实增长? | 胜率 | A/B/C/D |
| 2 | 卡在产业链关键环节? | 胜率 | A/B/C/D |
| 3 | 位置可能被市场重新理解? | 胜率 | A/B/C/D |
| 4 | 市场现在按普通公司还是瓶颈资产定价? | 胜率 | 普/瓶 |
| 5 | 判断错最多亏多少? | 盈亏比 | 止损% |
| 6 | 判断对有没有 5x/10x? | 盈亏比 | 倍数 |

**入场标准**:
- 胜率题至少 3 个 A/B
- 第 4 题答"普通公司定价" + 基本面已是瓶颈 = **认知差机会(最佳)**
- 5x+ 才考虑大仓位(5-10%);2-3x 只小仓位(3-5%)

## 认知差扫描清单(每周必做)

### A. 龙头大涨 → 上游/同链未涨
- 上游材料(CCL/电子布/光芯片/CMP 抛光垫)
- 同环节小厂(华天 vs 长电、新易盛 vs 中际旭创)
- 配套设备(封装/测试设备)

### B. 海外对标 → A 股映射
对标表存于 `knowledge/bottleneck-framework.md` "海外对标" section,实时更新。

### C. "误分类"标的扫描
- "传统化工"实为"AI 冷却液"(巨化)
- "电气设备"实为"AI 变压器"(金盘)
- "普通封测"实为"HBM 封装"(太极)

## 产业链下钻方法

某板块涨幅显著时,**不要停留板块级别**。拆解到子环节:

**芯片→封装→器件→模块→设备→测试→材料**

例:光模块板块不只看中际旭创,还要看光芯片(长光华芯)、测试设备(联讯)、光器件(天孚)、光纤(亨通)。

**产业集群效应**:苏州光子产业集群——龙头爆发后扫描同园区关联公司。
