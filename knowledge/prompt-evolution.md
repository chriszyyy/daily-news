### 📐 瓶颈资产投资框架（待集成到 CLAUDE.md）

**核心理念**：真正的10X机会不是"公司变好"，而是"市场重新理解一个公司在产业链里的位置"——从普通供应商→瓶颈资产，估值框架完全切换。

#### A. 升级 step 11（supply chain scan）— "三维度+瓶颈"扫描

每个supply chain环节必须回答3个问题：
1. **需求维度**：这个环节的需求是否真实增长？（订单/出货量/客户名单验证）
2. **供给维度**：这个环节是否被卡住？（产能扩张周期、技术门槛、原材料约束）
3. **认知维度**：市场是否还把它当作"普通供应商"，还是已经当作"瓶颈资产"定价？

**承压信号检测清单**（任一触发即标记为"潜在瓶颈"）：
- [ ] 交付周期延长（>12个月）
- [ ] 行业涨价（季度涨价>10%）
- [ ] 客户提前锁单/付预付款
- [ ] 头部公司议价权明显增强
- [ ] 产能扩张速度<需求增速

#### B. 升级 step 14（基本面深度研究）— 嵌入"6问框架"

对每个候选标的强制回答：
1. **真实增长**：所在产业是不是真实增长？（不是讲故事）
2. **关键节点**：是不是卡在产业链关键环节？
3. **认知差**：在产业链中的位置，市场是否可能重新理解？
4. **当前定价**：市场现在用什么估值框架（普通公司 vs 瓶颈资产）？
5. **风险**：判断错了最多亏多少？（明确止损位）
6. **赔率**：判断对了有没有5x、10x级别空间？

前4题判断**胜率**，后2题判断**盈亏比**。

#### C. 新增 step：估值框架转换扫描

对所有持仓和watchlist标的，每周扫描一次：
- 当前PE/PB所属估值框架（普通制造业10-20x / 周期股 / 成长股 / 瓶颈资产50x+）
- 是否出现"框架切换"催化剂（新订单、行业涨价、产业地位变化）
- 输出"重新定价潜力"评分：低/中/高

#### D. 升级仓位管理规则

**新规则**（覆盖现有规则）：
- 单票仓位 **3%-10%**（确定性高+核心瓶颈可至10%；弹性大未验证降至3%）
- **绝对禁止**单票>15%（避免情绪赌博）
- **必须**围绕一条主线平铺**至少3-5个核心环节**
- 持仓集中度警告：若任一标的>10%，下次操作必须减仓

**当前持仓审视**：
- 嘉实纳指 ¥15,208 (持仓占比~10.5%) ⚠️ 临界
- 黄金 ¥84,000+ (~58%) ❌ **严重超标**，但实物黄金性质特殊可接受
- 太极600667 ¥6,500 (~4.5%) ✅ 合理
- 工行 ¥11,800 (~8%) ✅ 合理

#### E. 新增"卖出验证清单"（替代"价格触发"卖出）

**卖出条件**（满足任一即考虑卖出，与价格无关）：
- [ ] 瓶颈不再是瓶颈（产能扩张完成/替代技术成熟）
- [ ] 需求被证伪（客户砍单/行业增速归零）
- [ ] 竞争格局变差（新进入者抢市场）
- [ ] 估值已透支未来3-5年（PE>历史90分位+增速放缓）
- [ ] 主线逻辑被推翻（如AI泡沫破裂）

**禁止卖出条件**：短期波动、情绪恐慌、看到别人涨

#### F. 新增"认知差扫描"任务（每周必做）

每周日扫描：
- 头部公司过去1月涨幅 vs 上游供应商涨幅 → 寻找"龙头大涨但上游滞涨"
- 头部公司ATH时，去其供应链找未充分定价的标的
- 关注"被市场误分类"的公司（如归类为周期股但实际成为AI瓶颈）

#### G. 升级 watchlist-sectors.md 结构

为每个板块添加3个标签：
- **位置**：核心瓶颈 / 关键节点 / 普通供应商 / 边缘
- **认知**：未被发现 / 部分定价 / 充分定价 / 过度定价
- **赔率**：10x+ / 5x / 3x / 2x / <2x

#### H. 实施优先级
1. **立即执行**：step 11升级（三维度+瓶颈检测）— 下次session即用
2. **立即执行**：6问框架嵌入step 14
3. **立即执行**：仓位规则更新（3-10%硬约束）
4. **本周内**：watchlist-sectors.md添加位置/认知/赔率三标签
5. **本周内**：决策日志加入"瓶颈定价"维度
