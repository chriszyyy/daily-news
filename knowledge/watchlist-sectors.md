## AI供应链全链条

### 瓶颈轮动规律
```
GPU/芯片 → 存储/HBM → 电力/变压器 → 液冷散热 → PCB/材料 → 先进封装 → 光模块/网络
```
瓶颈依次从上游向下游传导。当某环节供不应求时，关注该环节的定价权和业绩弹性。同时关注美股龙头暴涨后的A股映射标的。

### 分析方法
- **产业链下钻**：板块级别不够，要拆到子环节（芯片→封装→器件→模块→设备→测试→材料）对比涨幅找滞后
- **产业集群效应**：龙头爆发后扫描同城/同园区上下游（如苏州光子集群）
- **美股→A股映射**：美股供应链龙头暴涨后，寻找A股对应标的（如美光暴涨→A股存储映射；VRT暴涨→A股液冷映射）
